Items for Roger to load in his markup

There is a virtual function, presumably, called when the user enters the string: "Use <itemname
Keys --a key is disposable.  Using a key on a door sets the door's status to unlocked.  Is a derived class from item.

Crowbars --a crowbar is not disposable.  Using a crowbar on a crate or other crowbar-oriented item
	sets the crate's status to unlocked.  Is a derived class from item.
	
Treasure --derived class from item.  has a point value for that treasure.  Presumably, cannot be used.
